# Folder Dokumen Artefak

Silakan salin file-file dokumen artefak Anda ke dalam folder ini. 

Untuk memudahkan penautan pada website, berikut rekomendasi nama file untuk masing-masing artefak:

1. **RPP / Modul Ajar**: `rpp.pdf`
2. **Video Pembelajaran**: `video.mp4` (atau Anda bisa menggunakan link YouTube)
3. **Bahan Ajar**: `bahan_ajar.pdf`
4. **Media Pembelajaran**: `media_pembelajaran.pdf` (atau `.pptx` / link Google Slides)
5. **Lembar Kerja Mahasiswa (LKM)**: `lkm.pdf`
6. **Lembar Penilaian**: `lembar_penilaian.pdf`
7. **Lampiran 7 (Penilaian Perangkat)**: `lampiran_7.pdf`
8. **Lampiran 8 (Praktik Mengajar)**: `lampiran_8.pdf`

Setelah Anda meng-copy file ke folder ini, beri tahu saya agar saya dapat memperbarui tautan tombol "Lihat Detail" di portofolio Anda.
